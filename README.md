# LECTRA

Aplicación Flutter para Gestión de Reportes

## Versión
1.7.5

## Descripción
LECTRA es una aplicación móvil desarrollada en Flutter para la gestión de reportes.

## Características
- Pantalla de login con diseño moderno
- Colores: Azul oscuro (#1A237E) y Blanco
- Interfaz de usuario intuitiva

## Cómo ejecutar

### Prerrequisitos
- Flutter SDK instalado
- Android Studio o VS Code con extensiones de Flutter
- Un dispositivo Android o emulador

### Pasos
1. Instalar dependencias:
   ```bash
   flutter pub get
   ```

2. Ejecutar la aplicación:
   ```bash
   flutter run
   ```
